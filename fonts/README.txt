Timer fonts
===========

Place local timer fonts in this folder and select one in params.txt:

  timer_font_file=Roboto-Variable.ttf
  timer_font=Arial, sans-serif

Supported formats: WOFF2, WOFF, TTF and OTF. The file name must not include a
path. timer_font is the fallback CSS font list used when the local file is
missing, unsupported or cannot be loaded by the browser.

All bundled fonts have a plain, empty zero:

  Roboto-Variable.ttf          round colon; tabular digits
  OpenSans-Variable.ttf        round colon; tabular digits
  BarlowCondensed-Bold.ttf     round colon; compact tabular digits
  CourierPrime-Bold.ttf        monospaced; rectangular colon dots
  SyneMono-Regular.ttf         monospaced; round colon; angular digits

The bundled fonts are distributed under the SIL Open Font License 1.1. The
corresponding licence files are stored in this folder.

Шрифты таймера
==============

Поместите локальные шрифты таймера в эту папку и выберите один из них в params.txt:

  timer_font_file=Roboto-Variable.ttf
  timer_font=Arial, sans-serif

Поддерживаемые форматы: WOFF2, WOFF, TTF и OTF. Имя файла не должно содержать
путь. timer_font задаёт резервный список CSS-шрифтов, используемый, когда
локальный файл отсутствует, не поддерживается или не может быть загружен браузером.

Все входящие в комплект шрифты имеют обычный пустой ноль:

  Roboto-Variable.ttf          круглое двоеточие; табличные цифры
  OpenSans-Variable.ttf        круглое двоеточие; табличные цифры
  BarlowCondensed-Bold.ttf     круглое двоеточие; компактные табличные цифры
  CourierPrime-Bold.ttf        моноширинный; прямоугольные точки двоеточия
  SyneMono-Regular.ttf         моноширинный; круглое двоеточие; угловатые цифры

Входящие в комплект шрифты распространяются по лицензии SIL Open Font License 1.1.
Соответствующие лицензионные файлы находятся в этой папке.
